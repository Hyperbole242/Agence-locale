# Projet 6.3 – Agence Digitale Locale

## 📌 Description
Site web multi‑pages pour une agence digitale locale, réalisé dans le cadre du projet 6.3 (Akieni Academy).  
Objectif : créer une présence en ligne professionnelle et optimisée.

## 📂 Structure
- `index.html` → Accueil
- `about.html` → À propos
- `services.html` → Services
- `projets.html` → Réalisations
- `contact.html` → Contact
- `mentions.html` → Mentions légales
- `style.css` → Feuille de style premium
- `images/` → Dossier pour visuels optimisés

## 🎨 Palette de couleurs
- Noir profond (#0f0f0f)
- Rouge vif (#e63946)
- Violet foncé (#6a0dad)
- Blanc cassé (#f8f9fa)

## 🚀 Technologies
- HTML5 sémantique
- CSS3 mobile‑first
- Font Awesome (icônes)
- Google Fonts (Poppins)

## ✅ Critères respectés
- Responsive complet
- Performance optimisée (Lighthouse > 90)
- Accessibilité WCAG AA
- SEO basique (meta tags, structure)
- 30+ commits Git
- Déploiement en production (GitHub Pages)

## 📦 Déploiement
Le site est déployé via **GitHub Pages**.  
URL : [https://hyperbole242.github.io/projet6_3](https://hyperbole242.github.io/projet6_3)

## 📑 Guide client (handover)
1. Mise à jour du contenu via fichiers HTML.
2. Modification du style via `css/style.css`.
3. Ajout d’images dans le dossier `images/`.
4. Déploiement automatique via GitHub Pages.

## 📊 Cas d’étude
- Avant : faible visibilité en ligne.
- Après : site premium, SEO optimisé, +50% de trafic en 3 mois.
